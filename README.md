# DevNotes

> Qt / C++ / Python 学习笔记与技术知识库

记录日常学习中的技术笔记、代码片段、问题解决方案和最佳实践。

---

## 目录结构

```
DevNotes/
├── Qt/                    # Qt 开发相关
│   ├── C++/              # C++ 基础与进阶
│   ├── Python/           # PyQt / PySide
│   └── Qt6/              # Qt 6 新特性
├── SQLite/               # 数据库相关
└── Git/                  # 版本控制
```

---

## 内容分类

| 分类 | 说明 |
|---|---|
| **Qt/C++** | Qt 框架、信号槽、自定义控件、事件处理、QPainter |
| **Qt/Python** | PyQt6、PySide6、Python 桌面开发 |
| **Qt/Qt6** | Qt 6 新 API、迁移指南、性能优化 |
| **SQLite** | SQL 语法、数据库设计、Qt 中的 SQLite 使用 |
| **Git** | 版本控制、分支策略、协作流程 |

---

## 笔记格式

每篇笔记包含：
- 标题

- 代码示例

- 参考链接
